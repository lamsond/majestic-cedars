# FRQ Grading

## Attributes 10pts

* x,y, radius private (3)
* x,y radius instance (3)
* numCircles private (1)
* numCircles static (1)
* WIDTH, HEIGHT static final (2)
 
## Constructors 6pts

* correct signatures (2)
* default values to center (1)
* set x, y (1)
* set radius (1)    
* increment counter (1)

## calcArea() 5pts

* instance method (1)
* return type double(1)
* no parameters(1)
* correct calculation (1)
* return statement (1)

## calcPerimeter() 5pts

* instance method (1)
* return type double (1)
* no parameters (1)
* correct calculation (1)
* return statement(1)

## getters 8pts

* public methods (2)
* instance methods (2)
* correct return types (2)
* return statements (2)

## isOverlapping 6pts

* instance method (1)
* return type boolean (1)
* correct parameter (1)
* correct calculate distance (1)
* correctly check overlap (1)
* return statement (1)
